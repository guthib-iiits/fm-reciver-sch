# fm-reciver-sch
This is a project I saw in the book 101 things with Transistors and I was intrested in Radio communications. This one is a highly modified version of it. It uses a tank circuit and LTSpice inbuilt Frequency modulation to get the FM Signal and then the tank circuit is tuned using a pot and a variable capacitor. The signal is isolated from noise.
The isolated signal is then ampilfied using 5 amplifiers in series and then we can see the signal and test it's basic integrity by comparing the fiourier transform at input and output.
